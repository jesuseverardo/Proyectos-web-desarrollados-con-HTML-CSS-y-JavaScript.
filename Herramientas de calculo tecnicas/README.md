# Suite de herramientas técnicas

Aplicación web con calculadora de resistencias, asignación VLSM, diagramas de Bode, lugar geométrico de las raíces e historial guardado en `localStorage`.

## Tecnologías

HTML, CSS, JavaScript, Math.js y Chart.js.

Abre `index.html` con conexión a internet para cargar las bibliotecas externas.
