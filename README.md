# Proyectos Web con HTML, CSS y JavaScript

Este repositorio contiene una colección de proyectos web desarrollados con HTML, CSS y JavaScript.  
Los proyectos incluyen generadores, calculadoras y simuladores académicos enfocados en la práctica de desarrollo web y herramientas técnicas.

## Proyectos incluidos

- Generador de RFC
- Generador RFC base
- Herramientas de cálculo técnicas
- Simulador de frecuencias de un dado

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Objetivo

El objetivo de este repositorio es reunir diferentes prácticas y proyectos web desarrollados para fortalecer conocimientos en diseño de interfaces, programación en JavaScript y creación de herramientas interactivas.
